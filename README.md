# Web Project

This is a simple web project consisting of two main HTML pages:

1. **`form.html`** – A basic registration form.
2. **`website.html`** – A sample website preview demonstrating various HTML attributes and elements.

---

## Pages Description

# Registration Form

## form.html
- A simple registration form.
- Fields include:
  - First Name, Last Name
  - Email, Password
  - Date of Birth, Registration Date
  - Salary, Gender, Skills, Country
  - About You (textarea)
- Submit and Reset buttons included.
- Demonstrates basic HTML form structure and styling.

# Website Preview in Table

## preview.html
- Displays a website inside a table using an `<iframe>`.
- Table includes a header and a single row with the website preview.
- Demonstrates basic HTML table structure and embedding external websites.
- Example website: [Wikipedia](https://www.wikipedia.org)
